Code文件夹下命名规则如下：

experience2problem1buduichen2.m为[0,1]上拉格朗日二次插值；
experience2problem1buduichen5.m为[0,1]上拉格朗日五次插值；
experience2problem1buduichen10.m为[0,1]上拉格朗日十次插值；
experience2problem1duichen2.m为[-1,1]上拉格朗日二次插值；
experience2problem1duichen5.m为[-1,1]上拉格朗日五次插值；
experience2problem1duichen10.m为[-1,1]上拉格朗日十次插值；
即：文件名结尾数字表示拉格朗日插值次数；buduichen为[0,1]上插值；duichen为[-1,1]上插值。

experience2problem2.m为第二题预测所用程序。
